 Phase 3: Segmentation & Tracking — Two-Step Ingestor + Normalizer
  (v3 locked)

 Context

 The pipeline has metadata alignment (Phase 1-2) and frame
 contracts producing frame_manifest.csv. Phase 3 consumes that
 manifest, runs detector + tracker models per well, and emits
 canonical contract outputs. Construction 2 (Ingestor +
 Normalizer) isolates model-output parsing from contract mapping —
  model release drift hits ingestors only, contracts stay stable.

 All existing detection (gdino_detection.py), propagation
 (propagation.py), frame organization, and mask export code is
 reused as-is.

 Locked invariants

 - frame_index == time_int — both columns kept for backward
 compat, validated to match, sourced from frame_manifest.csv
 - Absolute pixel bbox is canonical — xyxy format, float values,
 clamped to image bounds
 - Parquet for intermediate contracts, CSV only for final
 segmentation_tracking.csv
 - Per-well shards first, merge to experiment-level — no write
 collisions
 - Provenance stamped once by NormalizeContext — not by ingestors
 - Deterministic sort before writes — (well_id, frame_index,
 image_id, embryo_id)
 - mask_type column on mask_rle — future-proofs yolk/aux mask
 integration
 - model_release defaults to "unknown" not "" — no empty-string
 join hazards

 Step 2: NormalizeContext (segmentation/normalize_context.py)

 Runner creates one per backend, stamps all records before
 normalization.

 @dataclass(frozen=True)
 class NormalizeContext:
     source_backend: str
     source_model: str
     model_release: str           # defaults to "unknown" if
 caller passes ""
     run_id: str

     def __post_init__(self):
         if not self.model_release:
             object.__setattr__(self, "model_release", "unknown")

     def stamp(self, records: list) -> list:
         """Apply provenance to all records in-place."""
         for r in records:
             r.source_backend = self.source_backend
             r.source_model = self.source_model
             r.model_release = self.model_release
             r.run_id = self.run_id
         return records

 Step 4: Ingestors (segmentation/ingestors/)

 __init__.py — dispatch + interface check

 REQUIRED_DETECTOR_FUNCTIONS = {"ingest_detections",
 "ingest_seed_selection"}
 REQUIRED_TRACKER_FUNCTIONS = {"ingest_propagation",
 "tracks_to_raw_masks"}

 def get_detector_ingestor(backend: str) -> ModuleType:
     mod = _import(backend)  # dispatch map
     missing = REQUIRED_DETECTOR_FUNCTIONS - {a for a in dir(mod)
 if not a.startswith("_")}
     if missing:
         raise AttributeError(f"{backend} ingestor missing:
 {missing}")
     return mod

 gdino_ingestor.py

 - ingest_detections(model, image_path, *, frame_index, image_id,
 image_height_px, image_width_px, ...) -> list[RawDetection]
   - Calls existing detect_embryos() + filter_detections()
   - Computes box_xyxy_abs from normalized + image dims, clamps to
  bounds
   - Sorts by descending confidence, then x_min, then y_min —
 defines deterministic detection ordering
   - Returns records without provenance (NormalizeContext stamps
 later)
 - ingest_seed_selection(frame_detections, *, experiment_id,
 well_id, video_id, ...) -> SeedSelection
   - Calls existing select_seed_frame()
   - Populates selected_detection_indices from sorted detection
 order

 sam2_ingestor.py

 - ingest_propagation(results, *, image_id_template,
 seed_frame_idx, ...) -> list[RawTrack]
   - Converts SAM2 {frame_idx: {embryo_id: {mask, bbox, area,
 confidence}}} to RawTrack
   - bbox values cast to float (canonical)
 - tracks_to_raw_masks(tracks, *, source_image_paths,
 exported_mask_dir, mask_type="sam") -> list[RawMask]
   - RLE-encodes via existing encode_mask_to_rle()
   - Computes centroids from numpy masks
   - Sets track.mask = None after RLE encoding to free memory
   - Sets mask_type from argument (default "sam")

 detectron2_ingestor.py / sam3_ingestor.py — stubs raising
 NotImplementedError

 Runner flow

 def run_segmentation_and_tracking(
     *, frame_manifest_csv, experiment_id, well_id, output_dir,
     pipeline_config, model_paths..., device, run_id, verbose
 ) -> dict[str, Path]:

     run_id = run_id or uuid.uuid4().hex[:8]
     backends = load_segmentation_backends_config(pipeline_config)

     # Load manifest, filter to well, enforce frame_index ==
 time_int
     well_df = ...

     # Create provenance contexts
     det_ctx = NormalizeContext(
         source_backend=backends.detector_backend,
         source_model="SwinT_OGC",  # or from config
         model_release=config.get("gdino_release", ""),  # ->
 "unknown" if empty
         run_id=run_id,
     )
     tracker_ctx = NormalizeContext(
         source_backend=backends.tracker_backend,
         source_model="hiera_large",
         model_release=config.get("sam2_release", ""),
         run_id=run_id,
     )

     # 1. Detect per frame -> stamp -> normalize -> write parquet
 shard
     raw_dets = [det_ingestor.ingest_detections(...) for frame in
 well_df]
     det_ctx.stamp(flat_dets)
     det_df = normalize_detections(flat_dets, ...)
     det_df.to_parquet(shard_dir / "frame_detections.parquet")

     # 2. Seed selection -> stamp -> write parquet shard
     seed = det_ingestor.ingest_seed_selection(...)
     seed.detector_backend = det_ctx.source_backend
     seed.run_id = det_ctx.run_id
     seed_df = normalize_seed_selection([seed])
     seed_df.to_parquet(shard_dir / "seed_selection.parquet")

     # 3. Track -> stamp -> normalize -> write parquet shard
     raw_tracks = tracker_ingestor.ingest_propagation(...)
     tracker_ctx.stamp(raw_tracks)
     tracks_df = normalize_tracks(raw_tracks, ...)
     tracks_df.to_parquet(shard_dir / "track_instances.parquet")

     # 4. Masks (RLE + PNG export) -> stamp -> normalize -> write
 parquet shard
     raw_masks = tracker_ingestor.tracks_to_raw_masks(raw_tracks,
 ...)
     # masks released from RawTrack by tracks_to_raw_masks (sets
 mask=None)
     tracker_ctx.stamp(raw_masks)
     masks_df = normalize_masks(raw_masks, ...)
     masks_df.to_parquet(shard_dir / "mask_rle.parquet")

     # 5. Final CSV contract
     final_df = build_segmentation_tracking_contract(masks_df,
 ...)
     final_df.to_csv(shard_dir / "segmentation_tracking.csv",
 index=False)

     # 6. Per-well validation sentinel
     _validate_shard(shard_dir)
     (shard_dir /
 ".segment_and_track.validated").write_text("validated\n")

 Key Design Decisions

 1. NormalizeContext centralizes provenance — ingestors return raw
  records without provenance. Runner stamps once per backend.
 Single attribution point.
 2. frame_index == time_int invariant enforced — both columns kept
  (backward compat), validated to match, sourced from
 frame_manifest.csv. Normalizers copy frame_index to time_int.
 3. Absolute pixel bbox canonical, xyxy float — normalized bbox
 kept only on RawDetection for traceability. All contracts use
 absolute pixel float coords.
 4. Parquet for intermediates, CSV for final — frame_detections,
 seed_selection, track_instances, mask_rle are Parquet
 (type-stable, handles RLE strings and lists well).
 segmentation_tracking.csv is the human-facing summary.
 5. Per-well shards + merge rule — prevents write collisions. Each
  Snakemake GPU job writes to per_well/{well}/. Merge rule
 concatenates validated shards to experiment-level.
 6. mask_type on mask_rle from day 1 — SAM2 writes
 mask_type="sam". When yolk/UNet masks arrive later, they write
 mask_type="yolk" etc. Unique key includes mask_type so they
 coexist.
 7. Deterministic detection_index — sorted by descending
 confidence, then ascending x_min, then ascending y_min. Stable
 across reruns given identical detections.
 selected_detection_indices on seed selection references these.
 8. model_release never empty — NormalizeContext defaults to
 "unknown". Prevents empty-string join hazards.
 9. Memory: mask freed after RLE — tracks_to_raw_masks() sets
 track.mask = None after encoding. Long wells don't balloon RAM.
 10. Normalizers split by file — one per contract, independent
 evolution.
 11. Module dispatch with interface check —
 get_detector_ingestor() asserts required functions exist at
 import time.
 12. Unique keys include (source_backend, run_id) — prevents rerun
  row duplication.
 13. Deterministic sort before all writes — (well_id, frame_index,
  image_id, embryo_id). Makes diffs reproducible.
 14. Per-well validation sentinels — merge consumes only validated
  shards. Experiment-level contract validated again after merge.
 15. Old schema untouched — REQUIRED_COLUMNS_SEGMENTATION_TRACKING
  stays as-is. New contracts use new constants. No legacy
 breakage.
