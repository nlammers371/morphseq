# Graphs

This family contains graph-structured phenotypic summaries and tree-style outputs.

High-level script sources:
- `10_principal_graph.py`
- `10_principal_tree.py`

Typical outputs:
- graph images and layout summaries
- tree candidates and branch statistics
- final graph / tree comparison figures

These folders are usually the outputs of downstream graph-fitting scripts rather than the raw condensation solver itself.
